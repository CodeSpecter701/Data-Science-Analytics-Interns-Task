🚀 Data Science & Analytics Projects

This repository contains multiple Machine Learning and Data Analytics projects developed using Python during hands-on practice and learning in Data Science & Machine Learning.

The projects focus on real-world datasets, exploratory data analysis, predictive modeling, and business insights generation.

📌 Project Summaries
Task 1 — Iris Dataset Analysis
Loaded and analyzed the Iris dataset
Created scatter plots, histograms, and box plots
Performed feature correlation analysis
Visualized species-based feature distributions
Task 2 — Credit Risk Prediction
Handled missing values using statistical methods
Encoded categorical variables
Built Logistic Regression and Decision Tree models
Evaluated models using Accuracy Score and Confusion Matrix
Task 3 — Customer Churn Prediction
Analyzed customer churn patterns
Performed categorical feature analysis
Visualized churn trends using count plots and histograms
Identified important customer behavior insights
Task 4 — Insurance Cost Prediction
Built a Linear Regression model
Processed and encoded insurance dataset features
Visualized the relationship between BMI, age, smoking, and charges
Evaluated predictions using MAE and RMSE
Task 5 — Bank Marketing Analysis
Performed customer demographic analysis
Built classification models for marketing prediction
Extracted feature importance insights
Identified factors influencing loan acceptance
